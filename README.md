# MusicBot
INCOMPLETE PROJECT  

## Installation
Download Node.js  
Clone this repo  
Run npm install  
Create a discord bot, more info here: https://discordpy.readthedocs.io/en/latest/discord.html  
Create a .env file and have one line of the form, TOKEN="YOUR_TOKEN_HERE"  

## Commands
$play <name>  
$pause  
$resume  
$stop
